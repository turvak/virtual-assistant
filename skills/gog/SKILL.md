# gog Skill

Description: Skeleton for a Gog.com (GOG) integration to assist with library catalog, game info, and purchase reminders. This is a starter scaffold to be fleshed out.

Purpose: Provide a starting point for a Gog-based assistant workflow (library listing, game details, purchases).

Triggers:
- "gog search <query>"
- "gog sync"
- "gog remind"

Payload:
- Kind: "systemEvent" or "agentTurn" (to be defined later). This is a scaffold.

Notes:
- This is a starter scaffold. Should be refined with actual API usage, authentication, and data structures.
